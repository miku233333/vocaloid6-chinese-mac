# Vocaloid 6 中文語音庫 - macOS 版

🎤 **Vocaloid 6 中文語音庫安裝腳本 (macOS)**

## 📦 內容

| 文件/文件夾 | 說明 |
|------------|------|
| `install.sh` | 一鍵安裝腳本 |
| `uninstall.sh` | 卸載腳本 |
| `data/` | 語音庫數據 |
| `docs/` | 文檔 |
| `scripts/` | 輔助腳本 |

## 🚀 安裝方法

### 方法 1: 一鍵安裝
```bash
chmod +x install.sh
./install.sh
```

### 方法 2: 手動安裝
1. 打開終端機
2. 導航到此目錄
3. 執行 `./install.sh`
4. 輸入管理員密碼

## ❌ 卸載

```bash
chmod +x uninstall.sh
./uninstall.sh
```

## 📋 系統要求

- **操作系統**: macOS 10.15+ (Catalina 或更新)
- **Vocaloid 6**: 已安裝
- **磁碟空間**: 至少 2GB
