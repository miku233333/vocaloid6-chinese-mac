# Private Translations

This public repository does not ship VOCALOID6-specific translation payloads.

Store your local translation files in:

- `~/.vocaloid6-chinese-mac-private/translations/zh-TW.json`
- `~/.vocaloid6-chinese-mac-private/translations/zh-CN.json`

You can also override the base directory with:

```bash
export VOCALOID6_PRIVATE_DATA_DIR=/your/private/path
```
