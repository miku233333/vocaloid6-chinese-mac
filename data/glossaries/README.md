# Private Glossaries

This public repository does not ship VOCALOID6-derived glossary data.

Put your local product-specific files here in your private workspace instead:

- `~/.vocaloid6-chinese-mac-private/glossaries/source-text-zh-TW.json`
- `~/.vocaloid6-chinese-mac-private/glossaries/nib-visible-zh-TW.json`
- `~/.vocaloid6-chinese-mac-private/glossaries/compiled-nib-zh-TW.json`

You can also override the base directory with:

```bash
export VOCALOID6_PRIVATE_DATA_DIR=/your/private/path
```
